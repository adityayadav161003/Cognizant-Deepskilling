# Angular Deepskilling

This module contains exercise solutions and configurations for learning Angular, covering core concepts from basic setup to state management and testing.

## Exercises Covered

1. **Environment Setup & Project Structure** (`Environment_Setup_Project_Structure`)
   - Angular CLI setup, configuration, and project architecture.
2. **Components & Data Binding** (`Components_and_Data_Binding`)
   - Component creation, template syntax, property binding, event binding, and two-way binding.
3. **Directives & Pipes** (`Directives_and_Pipes`)
   - Built-in directives (`*ngIf`, `*ngFor`), custom directives, built-in pipes, and custom pipes.
4. **Services & Dependency Injection** (`Services_and_Dependency_Injection`)
   - Injectable services, singleton patterns, hierarchical dependency injection, and data sharing.
5. **Routing & Navigation** (`Routing_and_Navigation`)
   - Router configuration, router links, route parameters, child routes, and route guards.
6. **Template Driven Forms & Validation** (`Template_Driven_Forms_and_Validation`)
   - Form controls, ngModel bindings, standard validation, and status tracking.
7. **Reactive Forms & Validation** (`Reactive_Forms_and_Validation`)
   - FormBuilder, FormGroup, FormControl, custom validators, and dynamic form fields.
8. **HTTP Client Integration & Interceptors** (`HTTP_Client_Integration_and_Interceptors`)
   - API consumption using HttpClient, REST requests, handling responses, and implementing HTTP interceptors for headers/logging.
9. **State Management with NgRx** (`State_Management_with_NgRx`)
   - Store configuration, actions, reducers, selectors, effects, and global state flow.
10. **Unit Testing with Jasmine & Karma** (`Unit_Testing_with_Jasmine_and_Karma`)
    - Component, service, and pipe testing, mock dependencies, and test bed configuration.
