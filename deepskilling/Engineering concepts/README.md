# Engineering Concepts Deepskilling

This module contains implementations of core Software Engineering concepts, specifically focusing on Algorithms, Data Structures, and common Design Patterns.

## Sections Covered

### 1. Algorithms & Data Structures (`Algorithms_DataStructures`)
Implementation of fundamental data structures and algorithmic paradigms to solve realistic business problems:
- **E-Commerce Platform Search**: Linear and binary search algorithms for product lookups.
- **Sorting Customer Orders**: Bubble Sort, Quick Sort, and Merge Sort implementation for sorting order data.
- **Inventory Management System**: Arrays and custom list data structures for item tracking.
- **Employee Management System**: Dynamic array/list data structures for employee record lookup and modification.
- **Library Management System**: Search and sort routines for book metadata.
- **Financial Forecasting**: Recursion vs. iteration optimization for predicting growth.
- **Task Management System**: Singly and doubly linked lists for task scheduling.

### 2. Design Patterns (`DesignPatterns`)
Implementations of GoF (Gang of Four) patterns in Java:
- **Creational Patterns**: Singleton, Factory Method, Builder.
- **Structural Patterns**: Adapter, Decorator, Proxy.
- **Behavioral Patterns**: Observer, Strategy, Template Method, MVC.
- **Pattern Concepts**: Dependency Injection.
