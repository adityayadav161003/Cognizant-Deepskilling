# Git Deepskilling

This module covers Git version control workflows, repository management, and best practices.

## Topics Covered

1. **Git Configuration & Basics** (`Git_Configuration_and_Basics`)
   - Initializing a repository, setting user identity, staging (`git add`), committing (`git commit`), and viewing history (`git log`).
2. **Git Branching & Merging** (`Git_Branching_and_Merging`)
   - Creating branches, switching branches (`git checkout` / `git switch`), merging branches (`git merge`), and tracking history with branch graphs.
3. **Git Conflict Resolution** (`Git_Conflict_Resolution`)
   - Simulating merge conflicts, inspecting conflict markers, manually resolving conflicts, and finalizing merges.
4. **Git Ignore Configuration** (`Git_Ignore_Configuration`)
   - Configuring `.gitignore` rules to exclude system files, IDE configurations, local env files, and binaries.
5. **Git Remote Operations** (`Git_Remote_Operations`)
   - Working with remote repositories using `git clone`, `git fetch`, `git pull`, and `git push`, and managing upstream branches.
