# Microservices Deepskilling

This module covers cloud-native architecture patterns using Spring Cloud, including service discovery, gateway routing, externalized configurations, and fault tolerance.

## Microservice Components Covered

1. **Config Server** (`Config_Server`)
   - Centralized configuration management using Spring Cloud Config Server, storing environment-specific property files.
2. **Eureka Service Discovery** (`Eureka_Service_Discovery`)
   - Service registry deployment where microservices register themselves and discover peer instances dynamically.
3. **Composite Microservices** (`Composite_Microservices`)
   - Designing composite backend services that orchestrate calls across multiple underlying microservices.
4. **API Gateway Routing** (`API_Gateway_Routing`)
   - Configuring Spring Cloud Gateway for centralized routing, request forwarding, path rewrites, and security filters.
5. **Resilience4j Circuit Breaker** (`Resilience4j_Circuit_Breaker`)
   - Implementing fault tolerance and resilience using Resilience4j circuit breakers, retries, and fallback methods.
