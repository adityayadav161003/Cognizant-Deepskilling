# React Deepskilling

This module contains exercise solutions for learning React, covering concepts from ES6 prerequisites to state management, routing, component lifecycles, and Jest/Enzyme testing.

## Key Exercises Covered

1. **Prerequisites & Basics**
   - **ES6 Cricket Players Map** (`ES6_Cricket_Players_Map`): Map and filter usage in ES6.
   - **React App Setup Welcome** (`React_App_Setup_Welcome`): Basic project instantiation and setup.
   - **JSX Office Rental CSS Rent** (`JSX_Office_Rental_CSS_Rent`): Creating components with JSX and custom styling.

2. **Components & Layouts**
   - **Functional Component Score Calculator** (`Functional_Component_Score_Calculator`): Simple state calculations in functional components.
   - **Multi Component Layout** (`Multi_Component_Layout`): Combining multiple smaller components into a larger page dashboard.
   - **CSS Module Cohort Dashboard** (`CSS_Module_Cohort_Dashboard`): Implementing CSS Modules for styled scoping.
   - **Props Shopping Cart** (`Props_Shopping_Cart`): Passing down data and callbacks through component properties.

3. **Interactivity & State**
   - **State Counter Mall Traffic** (`State_Counter_Mall_Traffic`): Complex local state and incrementers.
   - **Events Increment/Decrement/Currency** (`Events_Increment_Decrement_Currency`): Event listeners and inputs.
   - **Conditional Rendering Ticket Booking** (`Conditional_Rendering_Ticket_Booking`): Render elements conditionally.
   - **List Keys Blogger App** (`List_Keys_Blogger_App`): Rendering collections dynamically with standard React list keys.

4. **Forms & Advanced APIs**
   - **Forms Ticket Raising Reference** (`Forms_Ticket_Raising_Reference`): Form input bindings.
   - **Form Validation Mail Register** (`Form_Validation_Mail_Register`): Performing input validations prior to submission.
   - **Context API Employee Management Theme** (`Context_API_Employee_Management_Theme`): Implementing React Context API for shared dark/light theme options.
   - **Life Cycle Hooks Fetch Posts** (`Life_Cycle_Hooks_Fetch_Posts`): Lifecycle events and fetching data inside class components.
   - **REST API Random User Fetch** (`REST_API_Random_User_Fetch`): Dynamic fetch/axios calls for remote data.

5. **Routing & Testing**
   - **React Router Trainers List** (`React_Router_Trainers_List`): Dynamic navigation setup using React Router.
   - **Unit Testing Jest Enzyme Cohort** (`Unit_Testing_Jest_Enzyme_Cohort`): Component assertions using Jest.
   - **Mocking Axios Jest GitClient** (`Mocking_Axios_Jest_GitClient`): API isolation testing with mocked response data.
