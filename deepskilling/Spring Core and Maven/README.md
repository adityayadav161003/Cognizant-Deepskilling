# Spring Core and Maven Deepskilling

This module covers foundational Spring Core concepts, dependency injection mechanisms, bean configurations, Aspect-Oriented Programming (AOP), and Maven project structure.

## Exercises Covered

1. **Maven Multi Module Setup** (`Maven_Multi_Module_Setup`)
   - Creating multi-module Maven projects, setting up parent POMs, and managing module dependencies.
2. **Basic Spring Configuration** (`Basic_Spring_Configuration`)
   - Instantiating a Spring IoC container using XML configuration.
3. **Setter Dependency Injection** (`Setter_Dependency_Injection`)
   - Injecting dependencies using property tags and setter methods.
4. **Constructor Dependency Injection** (`Constructor_Dependency_Injection`)
   - Injecting dependencies using constructor-arg tags and constructors.
5. **Bean Scopes** (`Bean_Scopes`)
   - Demonstrating singleton and prototype bean scopes and their lifecycles.
6. **Bean Lifecycle** (`Bean_Lifecycle`)
   - Exploring custom init and destroy methods, and Spring lifecycle interfaces.
7. **Annotation AOP Logging** (`Annotation_AOP_Logging`)
   - Implementing logging aspects using AspectJ annotations (`@Before`, `@After`, `@Around`).
8. **XML AOP Logging** (`XML_AOP_Logging`)
   - Configuring Aspect-Oriented logging inside the XML application context.
9. **Exercise09** (`Exercise09`)
   - Comprehensive exercise consolidating bean configurations and dependency resolution.
