# Spring REST using Spring Boot Deepskilling

This module covers building RESTful APIs using Spring Boot, securing endpoints, validating user input, and implementing custom exception handlers.

## Exercises Covered

1. **REST Hello Endpoint** (`REST_Hello_Endpoint`)
   - Creating a basic Spring Boot application with a REST controller exposing standard GET endpoints.
2. **REST POST Endpoints** (`REST_POST_Endpoints`)
   - Processing JSON request bodies using `@PostMapping` and `@RequestBody`, and returning HTTP status codes.
3. **REST Custom Exception Handling** (`REST_Custom_Exception_Handling`)
   - Implementing global exception handling using `@ControllerAdvice` and `@ExceptionHandler` for custom error responses.
4. **REST Validation Errors** (`REST_Validation_Errors`)
   - Enforcing payload validation rules using Jakarta Validation annotations (`@NotNull`, `@Size`, etc.) and returning formatted validation errors.
5. **REST JWT Security** (`REST_JWT_Security`)
   - Integrating Spring Security, configuring JWT filters, generating JSON Web Tokens, and securing API endpoints.
