# Module 1 - Bootstrap 5 Up-skilling

This module covers the Bootstrap 5 responsive CSS framework, layout grids, components, utility classes, dynamic JS plugins, and custom Sass compiling.

## Exercises Covered

1. **Setting Up Bootstrap 5**
   - Incorporating Bootstrap via CDN and locally downloaded distribution files.
2. **Bootstrap Structure and Files**
   - Examining the library's pre-compiled CSS/JS file structure.
3. **Fundamentals of Responsive Grid Layout**
   - Understanding containers, rows, columns, and screen breakpoints (`xs`, `sm`, `md`, `lg`, `xl`, `xxl`).
4. **Column Layouts and Grid Classes**
   - Auto-layout columns, variable width content, and nesting grid rows.
5. **Alignment and Reordering in Grid**
   - Vertical/horizontal alignments and column reordering using `order-*` classes.
6. **Typography**
   - Headers, display titles, lead text blockquotes, and inline text elements.
7. **Colors and Backgrounds**
   - Applying contextual colors (`primary`, `secondary`, `success`, `danger`, `warning`, `info`, `light`, `dark`).
8. **Spacing Utilities**
   - Using margin and padding utility shorthand (e.g. `m-*`, `p-*`, `mx-*`, `py-*`).
   - **Borders, Shadows & Rounded Corners**: Border sizes, shadow depths, and corner styling.
9. **Display and Visibility**
   - Controlling element visibility using `d-none`, `d-block`, `d-flex`, and responsive display settings.
10. **Buttons**
    - Designing custom button styles, outlined buttons, sizes, and button groups.
11. **Cards and Media Objects**
    - Structuring content containers with headers, body panels, footers, and side-by-side lists.
12. **Navbars and Navigation**
    - Creating responsive headers, sidebars, drop-downs, and collapse components.
13. **Forms**
    - Layout controls for input groups, floating labels, selects, checkboxes, radios, and validation styles.
14. **Positioning Utilities**
    - Absolute, fixed, sticky, and relative position helper utilities.
15. **Responsive Flexbox Utilities**
    - Alignment, wrapping, and distribution of items inside flex containers (`d-flex`, `justify-content-*`, `align-items-*`).
16. **Icons with Bootstrap Icons**
    - Integrating vector SVG icons into web UI pages.
17. **Bootstrap 5 JavaScript Plugins**
    - Interactive controls like Modals, Tooltips, Popovers, and Carousels.
18. **Customization with Sass**
    - Overriding default Bootstrap variable defaults and compiling customized themes.
