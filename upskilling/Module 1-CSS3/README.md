# Module 1 - CSS3 Up-skilling

This module covers Cascading Style Sheets (CSS3), styling rules, layouts, typography, responsiveness, and web browser rendering diagnostics.

## Exercises Covered

1. **CSS Syntax and Comments**
   - Rulesets, selectors, properties, values, and writing inline, block, and multiline comments.
2. **Why CSS Inline vs Internal vs External**
   - Comparing styles loaded inline, inside document `<style>` headers, and via external `.css` links.
3. **Selectors Playground**
   - Practicing element, class, ID, attribute, pseudo-class (`:hover`, `:focus`), and pseudo-element (`::before`, `::after`) selectors.
4. **Color and Background Styling**
   - RGBA, HEX, HSL colors, backgrounds, gradients, and border-radius patterns.
5. **Box Model and Layout Control**
   - Margin, border, padding, height, width, box-sizing (`border-box`), and layout controls (`block`, `inline`, `inline-block`, `none`).
6. **Typography Fonts and Text**
   - Custom fonts, font weights, line heights, letter spacing, text alignment, and typography styling.
7. **Link and List Styling**
   - Designing custom hyperlinks, unordered/ordered lists, and hover animations.
8. **Table Styling**
   - Formatting data tables with padding, borders, cell collapse, alternate-row striping, and column width control.
9. **Multiple Columns in Text**
   - Implementing newspaper-style column layouts using CSS multicolumn properties.
10. **Responsive Web Design with Media Queries**
    - Constructing fluid, mobile-first responsive screens using CSS media queries (`@media`).
11. **Debug and Test with Dev Tools and VS Code**
    - Diagnosing styling priorities, inspecting the box model, and debugging stylesheets using live dev extensions.
