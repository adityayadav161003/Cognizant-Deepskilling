# Module 1 - HTML5 Up-skilling

This module covers fundamental and advanced HTML5 features, layout structures, forms, multimedia, APIs, and debugging techniques.

## Exercises Covered

1. **Create HTML5 Base Template**
   - Setting up a semantic HTML5 template with header, nav, main, section, article, aside, and footer tags.
2. **Welcome Message with Styling and ID Class**
   - Utilizing css selectors, IDs, and classes within a basic welcome page.
3. **Navigation and Linking**
   - Constructing relative and absolute navigation paths, anchors, and internal document linking.
4. **Image Gallery for Community Events**
   - Using image elements, captions, and responsive layout properties.
5. **Event Registration Form**
   - Form inputs, fieldsets, legends, placeholder values, labels, and various input types (text, email, tel, date).
6. **Video Invite with Media Events**
   - Embedding custom video content, media controls, fallback options, and scripting media events.
7. **Event Feedback with Events Handling**
   - Creating forms for feedback collection and checking event bindings.
8. **Saving User Preferences**
   - Leveraging LocalStorage/SessionStorage APIs to save user layout preferences on the client side.
9. **Geolocation for Event Mapping**
   - Integrating HTML5 Geolocation API to fetch user coordinates for mapping resources.
10. **Debugging with Chrome DevTools**
    - Analyzing elements, modifying attributes dynamically, and debugging DOM structures.
