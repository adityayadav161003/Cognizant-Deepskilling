# Module 1 - JavaScript Up-skilling

This module covers JavaScript (ES6+), functional programming concepts, DOM integration, event handling, asynchronous tasks, AJAX/Fetch APIs, jQuery, and testing.

## Exercises Covered

1. **JavaScript Basics and Setup**
   - Incorporating scripts (`<script>` tags, async/defer), console logging, and basic browser alerts.
2. **Syntax, Data Types, and Operators**
   - Declaring variables (`var`, `let`, `const`), working with primitive types, objects, and math/logic operators.
3. **Conditionals, Loops, and Error Handling**
   - If-else statements, switch-case logic, loops (`for`, `while`, `do-while`), and exception handling with `try-catch-finally`.
4. **Functions, Scope, Closures, & Higher-Order Functions**
   - Function declarations, arrow functions, lexical scope, lexical closures, and higher-order helper functions.
5. **Arrays and Methods**
   - Array creation, element indexing, and built-in iterators (`map()`, `filter()`, `reduce()`, `forEach()`, `find()`).
6. **Objects and Prototypes**
   - Object literals, constructors, prototypes, property descriptors, and JavaScript inheritance mechanisms.
7. **Modern JavaScript Features**
   - Utilizing ES6+ features like destructuring assignments, template literals, default arguments, and rest/spread operators (`...`).
8. **DOM Manipulation**
   - Querying elements (`querySelector`, `getElementById`), updating attributes, altering classes, and injecting markup dynamically.
9. **Event Handling**
   - Attaching event listeners (`addEventListener`), handling event propagation (capturing, bubbling), and custom event dispatching.
10. **Working with Forms**
    - Reading inputs, handling form submissions, custom validation logic, and managing error indications.
11. **Async JS (Promises & Async/Await)**
    - Managing asynchronous lifecycles using callbacks, ES6 Promises, and standard `async/await` syntax.
12. **AJAX and Fetch API**
    - Consuming external REST APIs using XMLHttpRequest and modern `fetch()` calls.
13. **jQuery and JS Frameworks**
    - Using jQuery selectors, DOM methods, animations, and exploring framework configurations.
14. **Debugging and Testing**
    - Inserting breakpoints, inspecting call stacks, using local logs, and executing simple assertions.
