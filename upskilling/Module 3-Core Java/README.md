# Module 3 - Core Java Up-skilling

This module contains 40 practical Java exercises covering language basics, Object-Oriented Programming (OOP) patterns, File I/O, advanced APIs, JDBC database connections, and modern concurrency features like Virtual Threads.

## Functional Areas

### 1. Basic Constructs & Operations
- **DataTypeDemonstration**: Variable ranges and primitive data types.
- **TypeCastingExample**: Narrowing vs widening reference casting.
- **OperatorPrecedence**: Logical, math, and relational operator precedence.

### 2. Control Flows & Decision Logic
- **SimpleCalculator** / **EvenOrOddChecker** / **LeapYearChecker** / **GradeCalculator**: Basic logical checks.
- **MultiplicationTable** / **FactorialCalculator** / **RecursiveFibonacci**: Dynamic loops and recursion.
- **NumberGuessingGame**: User input loops with java.util.Scanner.
- **PatternMatchingSwitch**: Java 21 preview switch statement matching.

### 3. Object-Oriented Programming (OOP)
- **ClassAndObjectCreation** / **MethodOverloading**: Core instance patterns.
- **InheritanceExample** / **InterfaceImplementation**: Polymorphic patterns and interfaces.
- **Records**: Java's immutable record types.
- **CustomException** / **TryCatchExample**: Robust error handling architectures.

### 4. Collections, Streams, & Lambdas
- **ArrayListExample** / **HashMapExample**: Using collections framework objects.
- **LambdaExpressions** / **StreamAPI**: Declarative pipeline filters and transformations.

### 5. File I/O & Networking
- **FileReading** / **FileWriting**: Streams for binary and text files.
- **HTTPClientAPI**: Fetching and parsing web payloads.
- **TCPClientServerChat**: Dynamic socket operations.

### 6. Concurrency & Multithreading
- **ThreadCreation** / **ExecutorServiceCallable**: Multi-worker setups.
- **VirtualThreads**: Lightweight Java 21 virtual threads.

### 7. JDBC Database Actions
- **BasicJDBCConnection** / **InsertUpdateJDBC**: Performing local SQL updates.
- **TransactionHandlingJDBC**: Manual transaction commits and rollbacks.

### 8. Introspection & Decompilation
- **ReflectionDemo**: Dynamically modifying attributes at runtime.
- **JavapBytecodeInspection**: Analyzing class structures.
- **DecompilerDemo**: Decompiling `.class` bytecodes using CFR.
